# 一些简单常用的github书写语法规则罗列在此
### 标题
```
# the largest heading
## the second largest heading
###### the smallest heading
```
# the largest heading
## the second largest heading
###### the smallest heading



### 字体样式
加粗 `** ** or __ __`

**this is bold text**

斜体 `* * or _ _`

*this text is italicized*


### 引用文本
`> text`
> pardon my French

### 引用代码
`code is here`
```
git status
```

### 链接
`[text](url)`

[GitHub Pages](https://pages.github.com)
